面试题:
  L = [1, 2, 3]
  def f(n, lst=[]):
      lst.append(n)
      print(lst)
  f(4, L)  # 请问结果是[1, 2, 3, 4]
  f(5, L)  # 请问结果是[1, 2, 3, 4, 5]
  f(100)   # 请问打印结果是[100]
  f(200)   # 请问打印结果是[100, 200]
说明:
  默认参数(缺省参数)绑定在函数对象内部，具随函数的生命一直存在

解决办法:
  L = [1, 2, 3]
  def f(n, lst=None):
      if lst is None:
          lst = []
      lst.append(n)
      print(lst)
  f(4, L)  # 请问结果是[1, 2, 3, 4]
  f(5, L)  # 请问结果是[1, 2, 3, 4, 5]
  f(100)   # 请问打印结果是[100]
  f(200)   # 请问打印结果是[200]
